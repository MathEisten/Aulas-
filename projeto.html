Inspetor de elementos

O inspetor de elementos é uma ferramenta essencial para desenvolvedores web, usada para examinar e modificar o código de uma página da web diretamente no navegador.

Ele faz parte das ferramentas de desenvolvedor (DevTools) disponíveis em navegadores como Google Chrome, Firefox, Edge e Safari.

Html, oque é?

O HTML "hipertext markup language" uma caracteriza-se por ser uma linguagem de marcação que define a estrutura de uma página web, Uma linguagem que indica aos navegadores como apresentar conteúdos, como texto, imagens, cabeçalhos e outros elementos multimédia.

O HTML foi criado por Tim Berners-Lee, o inventor da World Wide Web, com o objetivo de facilitar a comunicação e disseminação de documentos entre pesquisadores.

A primeira versão do HTML foi publicada em 1991. O HTML não é considerado uma linguagem de programação, já que ele não pode criar funcionalidades dinâmicas. Ao invés disso, com o HTML, os usuários podem criar e estruturar seções, parágrafos e links usando elementos, tags e atributos. Os documentos HTML são arquivos de texto simples, com a extensão .html, e podem ser escritos na maioria dos editores de texto comuns, como o Bloco de Notas, do Windows.

Em aplicações mais avançadas, recomenda-se a utilização de ambientes de desenvolvimento integrado, como o Visual Studio Code, que oferecem recursos que facilitam a escrita do código.

Funcionamento das tags

As tags no HTML servem para estruturar, organizar e definir o conteúdo de uma página web. Elas informam ao navegador como os elementos devem ser exibidos e qual o propósito de cada parte do documento.

Cada tag tem uma função específica e contribui para a organização e apresentação do conteúdo de forma eficiente e padronizada. Tags semânticas ajudam navegadores e leitores de tela a interpretar o conteúdo.

Tags de texto

O HTML permite organizar textos de forma hierárquica. As tags de cabeçalhos, como h1 e h6, são usadas para títulos e subtítulos, sendo "h1" o mais importante e "h6" o menos relevante.

Para parágrafos, usamos a tag "p", enquanto "b"(ou strong) e "i" (ou "em") são utilizadas para destacar partes do texto em negrito ou itálico, respectivamente. A tag "br" é utilizada para inserir quebras de linha

Tags para links e imagens

Links são criados com a tag "a", que utiliza o atributo href para definir o endereço de destino.

Já para adicionar imagens, usamos a tag "img", que é uma tag de auto fechamento. O atributo src especifica o caminho da imagem, enquanto alt fornece uma descrição para leitores de tela
